
# 📘 Assignment: Jogo da Forca

## 🎯 Objective

Construir um jogo da Forca em Python para praticar manipulacao de strings, loops, condicionais e selecao aleatoria.

## 📝 Tasks

### 🛠️ Implementar logica do jogo

#### Description
Crie um programa de jogo da Forca que selecione uma palavra aleatoria e permita ao jogador adivinhar letras ate descobrir a palavra ou acabar o numero de tentativas.

#### Requirements
Completed program should:

- Selecionar palavras aleatoriamente de uma lista predefinida
- Aceitar palpites de letras e mostrar o progresso atual no formato `_ _ _`
- Rastrear tentativas incorretas restantes
- Encerrar quando a palavra for adivinhada ou as tentativas forem esgotadas
- Exibir mensagens finais de vitoria ou derrota
